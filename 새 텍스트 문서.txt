자료형의 크기
byte 1byte
short 2byte
int 4byte
long 8byte
float 4byte
double 8byte
boolean 1bit
char 2byte


형변환(타입변환)
자동형변환(묵시적형변환)
작은 자료형에서 큰 자료형으로 바꿀때
정수 -> 실수
문자 -> 정수

강제형변환(명시적형변환)
(원하는 자료형)데이터 or 변수
큰 자료형에서 작은 자료형으로 바꿀때
실수 -> 정수 (소수점 이하 부분이 소실된다.)
정수 -> 문자

데이터의 입력
java.util에 있는 Scanner 클래스를 사용해야된다.

inport
-다른 패키지에 있는 클래스를 사용하겠다고 명시하는것

import java.util.Scanner;

클래스 안에는 함수(기능)가 정의되어 있다.

클래스 안에 정의된 메서드를 사용하기 위해서는 객체를 생성해야한다.

객체를 생성하는 방법(new 클래스명();)
new<-메모리에 공간확보 클래스명();<-확보된공간에 객체생성
int x = 100;
char c = 'a';
Scanner sc = new Scanner(System.in);

System.in -> 키보드 입력을 받는 기본 통로

만들어진 객체를 통해서 클래스 안에 정의되어 있는 메서드를 호출할수있다.

sc.메서드명();

연산자

산술연산자 : +, -, *, /(몫), %(나머지)
비교연산자 : == ,!= ,< ,<= ,> ,>=
논리연산자 : &&, ||, !
증감연산자 : ++, --
비트연산자 : &, |, ^, ~
시프트연산자 : <<, >>, <<<, >>>

-if - else -> 참일때 실행할 명령, 거짓일때 실행할 명령
-if -else if -> 여러조건을 비교하여 실행할 명령을 결정
