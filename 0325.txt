제어문(코드의 흐름을 제어하는 문법)

기본적인 코드의 흐름
위에서 아래로, 좌에서 우로 동작한다.(대입연산자는 오른쪽에서 왼쪽으로 동작)

조건식 : 연산의 결과가 true 또는 false로 반환되는 식
(비교연산자,논리연산자)

조건문(분기문)
조건에 따라 실행할 명령을 결정하는 문법

if문(범위로 비교를 할 때)
- if문
- if -else
- if - else if

switch문(값을 1대1로 비교할 때)
- 비교값과 일치하는 조건값의 case로 접근하여 바로 실행

java 12부터는 switch를 연산식(expression)으로 사용할  수 있다.

반복문
조건에 따라 명령을 반복할지 말지 결정하는 문법

for문 - 반복횟수가 정확하게 정해져 있을때
while문 - 반복횟수를 정확히 모를때도 사용할 수 있다.

기타제어문
현재 반복문은 조건이 거짓이 될 때까지 반복을 한다.
내가 원하는 조건에 도달하면 반복횟수가 남았더라도 반복문을 끝내거나, 다음횟수도
건너뛰게 만들수있다

break
반복문 안에 break를 만나게 되면 그 즉시 반복문을 종료한다.

continue
반복문 안에서 continue를 만나게 되면 증감식으로 간다.
