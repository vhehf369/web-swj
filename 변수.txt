변수 : 데이더를 담는 상자다
프로그램에서 데이터를 저장하기 위해 메모리 공간에 이름을 부여한것

자료형 : 데이터의 재질과 크기
변수에 저장할 데이터의 종류와 크기를 정의하는 것

종류
정수형
byte		1byte
short		2byte
int 		4byte
long		8byte

변수를 만드는 방법
1. 변수의 선언

자료형 변수명; -> 값이 들어있지 않은 상태

2. 변수에 값을 대입

변수명 = 값;

대입은 원하는 만큼 가능하다.
대입을 하면 이전의 값은 사라진다.

3. 변수의 초기화(initialize)

자료형 변수명 = 값;

초기화 후 대입이 가능하다.

기본자료형
정수형

참조자료형
기본자료형 -> 값 자체를 들고 있다.
참조자료형 -> 값이 있는 주소값을 알고 있다.

형변환

자동형변환(묵시적형변환)
1. 작은 자료형 -> 큰 자료형
2. 문자형 데이터 -> 정수

강제형변환(명시적형변환)
2. 큰 자료형 -> 작은자료형
3. 정수 -> 문자
